# 2026-07-05 — Banner redesign (v2)

## Banner SVG

- **Action:** Created assets/banner.svg (7.6KB, generated by seeded script in session scratchpad); README.md reduced to one linked banner image.
- **Why:** Brief: delete all, single clickable banner → meo.in.th. Dark purple "luxury" theme, faint grid, low-opacity boxes clustered like a nebula (2 gaussian clusters + sparse field, seeded LCG for reproducibility). Font switched Hack → CMU per user; SVG in GitHub `<img>` cannot load external fonts, so font stack is `'CMU Serif' → 'Latin Modern Roman' → Georgia → serif` — viewers without CMU see Georgia.
- **Result:** Rendered preview via qlmanage twice. v1 bug: nebula ellipses bled outside viewBox in non-browser renderers and grid was invisible (opacity 0.07). Fixed with explicit `clipPath` + grid opacity 0.12.
- **Lesson:** 📌 GENERAL: SVG viewBox clipping is browser behavior, not SVG-file behavior — some renderers (qlmanage, some previewers) overflow. Explicit clipPath costs one line and removes the whole renderer-dependence class of bugs.
